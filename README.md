# React app setup

**Steps to run on local**

- Clone repo
- Do `npm i` to install depedencies
- To start Server `npm run dev`
- server will start on `http://localhost:5173/`

**Steps to run as production build**

- run `npm run serve`
- server will start on `http://localhost:3000/`
